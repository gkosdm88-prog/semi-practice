# semi-practice

세미님을 위한 연습용입니다.

코드 수정 → 커밋 → 푸시 연습용 레포입니다.

## 🚀 가장 쉬운 수정 방법 (github.dev)

1. GitHub에서 이 레포 페이지를 엽니다 → https://github.com/jinnyjiinlee/semi-practice
2. 키보드에서 **`.` (마침표)** 키를 누릅니다.
3. VS Code와 똑같은 화면이 브라우저에서 열립니다.
4. 왼쪽 파일 목록에서 `index.html` 클릭.
5. `<!-- ✏️ -->` 표시된 부분의 글자를 원하는 대로 수정합니다.
6. 왼쪽 사이드바에서 **Source Control 아이콘** (가지 모양) 클릭.
7. 메시지 칸에 변경 내용 적기 (예: `제목 수정`).
8. **Commit & Push** 버튼 클릭. 끝! ✅

## 🌐 결과 미리보기

GitHub Pages를 켜면 수정한 내용이 바로 웹사이트로 보입니다.

- Settings → Pages → Source: `Deploy from a branch` → Branch: `main` / `/ (root)` → Save
- 1~2분 후 https://jinnyjiinlee.github.io/semi-practice/ 에서 확인

## 📝 어디를 수정하나요?

`index.html` 파일에서 `✏️` 이모지가 있는 줄을 찾아 그 아래 텍스트를 바꾸면 됩니다.

- `<title>` → 브라우저 탭에 보이는 제목
- `<h1>` → 큰 인사말
- `<p>` → 본문 문장
